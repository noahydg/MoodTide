# MoodTide 心潮 · 设计定稿（结构基准）

> 本文 = 团队确认的产品结构。改结构先改这里，代码随之对齐。
> 赛道三「刷到懂你的瞬间」· 移动端 Web H5（抖音视觉风格）。

---

## 0. 从人性出发：我们在满足什么

人对情绪/音乐产品的四个深层需求，MoodTide 每一处都要对上：

| 人性需求 | 产品兑现 |
|---|---|
| **被看见**（说不清心情，却渴望被懂） | 客观推断主观 + 卡片亮出“为什么是这首” |
| **不孤独**（原来此刻这样的不止我） | 共鸣计数“今夜 N 人刷到相似的潮” |
| **独一无二**（只属于这一刻的我） | 每段不重复 + 刷走即焚 + 长按收下 |
| **掌控感**（没全懂时我能纠正） | 第二层“懂我/不太对”微反馈 + 情绪轮微调 |

情绪用心理学的二维坐标承载：**愉悦度(valence) × 唤醒度(arousal)**。
每种心情是坐标上的一个点 → 客观信号融合出坐标 → 坐标决定旋律/配色/文案。

## 2. ① 情绪推断引擎 MoodSense（灵魂）

**客观因素 → 主观心境**。信号按“有多私人”分三层，越往下越懂你：

| 层 | 信号 | 代码 | 性质 |
|---|---|---|---|
| 环境层 | 时间、天气、季节/节气 | `sense/signals.ts` | 真实、稳、零门槛 |
| **处境层** | **你刚刷过什么**（迷你信息流） | `feed/prerollFeed.ts` | **抖音独有的“懂你”，权重最高** |
| 个人层 | 定位、历史潮汐瓶 | （远期，涉隐私） | 暂不做 |

**管线**（`sense/moodSense.ts`）：

```
采集信号(各带 valence/arousal/weight/reason)
   → 加权融合 → 心境坐标 {valence, arousal}
   → nearestMood() snap 到情绪档(选曲/配色)        ← 混合模型：内部连续，对外几档
   → moodLabel() 心境标签 / 拼“为什么是这首” / 共鸣计数
   → MoodInference  (驱动一张卡片)
```

**推断时机**：在卡片 `enter()`（刷到那一刻）才跑，这样能读到“你刚刷过什么”。

### 迷你信息流（处境层落地，团队拍板方案）

黑客松拿不到抖音真实浏览数据 → **自建一小段信息流**：评委进 MoodTide 前先刷
2-4 张我们自己的内容卡（`feed/contentCard.ts`，数据见 `PREROLL`），每张打好情绪标签。
刷到 MoodTide 时读“你刚停在了哪几张”→ 推断。

> 三重收益：① 推断是真的、现场可验证；② 证明“卡片就在信息流里”（赛道三要点）；
> ③ 演示可控——换前置内容就能现场演示“不同处境→不同的歌”。

现场演示例：评委刷过「雨夜便利店」「凌晨改方案」→ MoodTide 说
**“深夜 · 雨 · 你刚停在了雨夜与独处”** → 起一段深夜微忧的潮。可信、不是魔术。

## 3. ② 第一层·即时卡片（怎样吸引你进来）

**第一眼极简**（音乐 + 声波海面 + 一句读心文案），其余信息渐显/置于边角，**不堆**。

卡片四件套（`feed/card.ts`，全部团队确认要做）：

| 件 | 内容 | 满足 |
|---|---|---|
| 基底 | 刷到即响 + 声波海面 + 读心文案 + 长按收下 | 刷到即懂 / 独一无二 |
| ① 为什么是这首 | 一行推断依据「深夜 · 雨 · 你刚停在了雨夜与独处」 | 被看见（可信） |
| ② 心境坐标/标签 | 标签「低落偏平静」+ 右上角坐标点 mini-map | 被看见（可感） |
| ③ 懂我 / 不太对 | 微反馈：懂我→加深认同；不太对→桥接第二层微调 | 掌控感 + 数据 |
| ④ 共鸣计数 | 「今夜 N 人刷到相似的潮」 | 不孤独 / 传播 |

**手势消歧**（`interaction/holdToKeep.ts`）：轻点→进详情 / 长按→收下 / 拖动→滚动。

**纪律**：第一层**永远零延迟、绝不 loading**——只用合成器/预生成 mp3，不在这里调在线生成。

---

## 4. ③ 第二层·点进去（给你掌控感）

`detail/detailView.ts`。轻点任意心潮卡进入，承载“功能点都 cover”：

- 看完整推断（contributors 可展开“为什么”）
- **情绪轮 / 拨心境坐标微调** → 旋律当场变形（它没全懂时你来纠正）
- ✨ **豆包音乐真实生成**一首更精致专属曲（异步，有兜底，仅这里触发在线）
- 收下 / 命名 / 加入潮汐瓶
- 生成分享卡（竖屏 9:16 + 情境水印 + 二维码）

---

## 5. 音乐与版权（团队定稿）

- 主路径：**豆包音乐生成模型**（火山 Seed-Music/豆包·音乐），仅第二层主动触发。
- 四级降级链（`music/musicService.ts`）：**豆包音乐 → Suno 代理 → 预生成 mp3 池 → 合成器（永不挂）**。
  第一层只走后两级零延迟来源。
- 音源版权：**Suno 生成**，分享物注明“AI 生成”+ MoodTide 水印。
- API Key 只在后端 `server/index.ts`(BFF)，前端只调 `/api/*`。
