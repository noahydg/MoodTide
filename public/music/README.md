# MoodTide 预设音乐（豆包生成）

把豆包(或即梦/海绵)生成的 **4 段循环纯音乐** 放进本文件夹，文件名必须如下，
应用会自动加载、循环播放；放进来即生效，无需改代码。

| 文件名 | 心境 | 建议时长 | 调性/速度 |
|---|---|---|---|
| `mood0.mp3` | 深夜·睡不着 | 20–30s loop | F 小调 · ~58 BPM |
| `mood1.mp3` | 黄昏·一个人 | 20–30s loop | C 大调 · ~68 BPM |
| `mood2.mp3` | 通勤·走神 | 20–30s loop | A 小调 · ~75 BPM（带律动） |
| `mood3.mp3` | 雨天·想念 | 20–30s loop | D 小调 · ~60 BPM |

## 导出建议（控制 ZIP ≤8MB）
- 格式 mp3，码率 96–128kbps，单段 ~20–30 秒、可无缝循环。
- 4 段合计约 1.5–2.5MB，提交包总额度够用。
- 纯音乐、**无人声**、**无突兀鼓点**（疗愈调性，要的是“律动感/呼吸感”不是“蹦迪”）。

## 节奏（不同频率）说明
拍海面 / 速度滑杆会用 `playbackRate` 微调速度（±约20%以内变速，音高变化可接受）。
若要“变速不变调”，需更复杂的时间拉伸，demo 阶段先用变速即可。

放好文件后刷新页面 → 「查看详情」→ 点任意卡，即播放对应的豆包音乐。
文件缺失时自动回退到内置合成音（保证 demo 不静音）。
